# Widgets

The widget collection spans small convenience controls and larger
application-oriented components. The pages below focus on the widgets that are
already documented with examples and API notes.

## Selection And Input

- [QtToggleGroup](./qt_toggle_group.md): compact toggle button groups with
  exclusive and multi-select modes

## Disclosure And Layout

- [QtCheckCollapsible](./qt_collapsible.md): collapsible sections with an
  optional checkbox, action button, and warning indicator
- [QReadMoreLessLabel](./qt_label_read_more.md): rich-text content that can
  expand from a short summary into a full explanation

## Progress And Status

- [QtActiveOverlay](./qt_active_overlay.md): animated activity indicators
- [QtActiveProgressBarButton](./qt_button_progress.md): button with integrated
  progress state
- [QtOverlay](./qt_overlay.md): floating labels and inline messages attached to
  another widget
- [QtStepProgressBar](./qt_progress_step.md): step-based progress indicator for
  multi-stage workflows
